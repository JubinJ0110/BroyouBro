  • Visual Studio Code
  • RapidAPI
  • Time, Datetime and requests Library

Feature List:

  • Gives back Stock Information

Future Feature List:

  • Ask stock related questions

Basic Technical Features: Use of python

Advanced Technical features: API calls, pip3, performance counter timer

What was Originally Planned:

Have a Python application that could retrieve stock data for that day when the user entered a stock ticker symbol

What was Delivered in my Final Project:

A Python Application that can return 5-6 months of daily stock data and 20 years worth of weekly stock data when the user inputs a stock ticker symbol. Shows the time and date, how long the user has spent in the application and how long it took to get the results. I also had it loop through aking for stock information. Made sure there were error messages when the user enters an invalid input.

What Features I'd add with more time:

I'd use pandas to maybe try and organize the data better. Or I would try and figure out tkinter to maybe make it look more aesthetically pleasing.
